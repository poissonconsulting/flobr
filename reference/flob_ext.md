# Flob Extension

Flob Extension

## Usage

``` r
flob_ext(flob)
```

## Arguments

- flob:

  A
  [`flob()`](https://poissonconsulting.github.io/flobr/reference/flob.md)

## Value

A string of the extension of the original file.

## See also

[`flobr()`](https://poissonconsulting.github.io/flobr/reference/flobr-package.md)
and
[`flob_name()`](https://poissonconsulting.github.io/flobr/reference/flob_name.md).

## Examples

``` r
flob_ext(flob_obj)
#> [1] "pdf"
```

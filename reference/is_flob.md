# Is flob

Is flob

## Usage

``` r
is_flob(x)
```

## Arguments

- x:

  An object to test

## Value

A flag indicating whether x inherits from
[`flob()`](https://poissonconsulting.github.io/flobr/reference/flob.md).

## See also

[`flobr()`](https://poissonconsulting.github.io/flobr/reference/flobr-package.md)

## Examples

``` r
is_flob(flob_obj)
#> [1] TRUE
```
